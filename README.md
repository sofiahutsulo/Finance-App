# Finance App

Android-додаток для управління особистими фінансами, який я розробила на Kotlin.

## Опис

Я створила мобільний додаток для відстеження доходів, витрат та планування бюджету. Додаток дозволяє керувати рахунками, аналізувати фінансову статистику та контролювати витрати за категоріями.

## Технології

**Мова програмування:** Kotlin

**Архітектура:** MVVM + Clean Architecture

**UI:** Jetpack Compose + Material 3

**Dependency Injection:** Hilt

**База даних:** Room Database

**Мережа:** Retrofit 2 + OkHttp

**Асинхронність:** Kotlin Coroutines + Flow

**Навігація:** Navigation Compose

**Локальне зберігання:** DataStore

## Що я реалізувала

### Екрани
- **Авторизація** — реєстрація та вхід користувача
- **Головна** — баланс, статистика за місяць, останні транзакції
- **Транзакції** — список всіх операцій з фільтрацією
- **Рахунки** — управління банківськими рахунками та картками
- **Бюджети** — планування та контроль бюджету за категоріями
- **Статистика** — графіки та аналіз витрат (діаграми, bar charts)
- **Профіль** — налаштування користувача

### Функціональність
- Додавання доходів та витрат
- Категоризація транзакцій
- Управління кількома рахунками
- Встановлення бюджетів
- Візуалізація статистики через власні компоненти графіків
- Синхронізація з сервером через REST API

## Структура проєкту

Я створила таку структуру:

```
app/src/main/java/com/finance/manager/
├── data/
│   ├── local/          # Room Database, DAO, Entity класи
│   ├── remote/         # API Service, DTO
│   └── repository/     # Repositories для роботи з даними
├── domain/
│   └── model/          # Domain моделі
├── ui/
│   ├── components/     # Reusable UI компоненти
│   ├── screens/        # Екрани додатку
│   ├── navigation/     # Навігація між екранами
│   └── viewmodel/      # ViewModels
├── di/                 # Hilt модулі для DI
├── FinanceApp.kt       # Application клас
└── MainActivity.kt     # Головна активність
```
